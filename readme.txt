
------------------------------
        Overview :- 
------------------------------


This Script will generate textfiles containing the basic insight of your followers and following of your instagram account .
This Script will return username of insgram accounts.


------------------------------
        Output :- 
------------------------------

1) Text file containing list of your followers.
2) Text file containing list of your followings.
3) Text file containing list of users whom you follow and who also follow you back.
4) Text file containing list of users whom you follow but they don't follow you back.
5) Text file containing list of userd who follows you but you don't follow them back.


------------------------------
      Requirement :-  
------------------------------
Python
Selenium
webdriver ( i have user chromedriver , you can use any other )


------------------------------
        NOTE :- 
------------------------------

please turn off two-step verification in your instagram account
if you don't want to turn off you can modify the script also
